# 📊 Data Analyzer (Python)

## 📌 Project Overview

**Data Analyzer** is a menu-driven Python application that allows users to enter numbers and perform different operations on them. It is designed to practice Python fundamentals such as functions, recursion, lambda functions, sorting, `*args`, `**kwargs`, global variables, and docstrings.

---

## 🎯 Features

* ✅ Input multiple numbers
* ✅ Display data summary

  * Total Elements
  * Minimum Value
  * Maximum Value
  * Sum
  * Average
* ✅ Find factorial using recursion
* ✅ Filter numbers using Lambda Function
* ✅ Sort data

  * Ascending
  * Descending
* ✅ Display statistics
* ✅ Demonstration of `*args`
* ✅ Demonstration of `**kwargs`
* ✅ Uses Global Variable
* ✅ Uses Docstrings

---
